## Usage
This model aims to compare job descriptions against a collection of reumes, and rank them using cosine similarity and TF-IDF.


## Requirements:
```
    pip install pdfminer.six
    pip install docx2txt
    pip install nltk
    pip install numpy
    pip install gensim
    pip install python-Levenshtein
    pip install pandas
```
